Desarrollador
Fullstack

Un desarrollador full stack es un profesional que domina tanto el desarrollo frontend (la parte visual que interactúa el usuario) como el backend (la lógica y servidores que operan detrás de escena) de una aplicación o sitio web, siendo capaz de crear y mantener todo el ciclo de desarrollo de software.

Acerca de mi

Soy Diego Fernando Mariño Cruz, Graduado del Intituto AIEP de Ingenieria de Ejecucion en Informatica, con experiencia en tecnologias como Vue.js PL/SQL, Postman, Java, JavaScript, etc.

Vue.js

Framework progresivo de JavaScript de código abierto para construir interfaces de usuario (UIs) y aplicaciones web.

Java Script

Lenguaje de programación que hace las páginas web interactivas, añadiendo animaciones, actualizaciones de contenido y efectos dinámicos.

PL/SQL

Es la extensión procedimental de Oracle para SQL, un lenguaje de programación de bases de datos que combina sentencias SQL con la capacidad de implementar lógica de procedimiento como bucles y condicionales
